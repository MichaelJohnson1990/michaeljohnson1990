
Please Notes:

This example shows the usage of:

		* The user plugin
		* The auth plugin
		* The admin plugin
		* The data-editor plugin

Run with:

$ node app.js

Or to use a custom port:
$ node app.js --seneca.options.main.port=4000

Then visit:
http://localhost:3000

Access the admin panel on:
http://localhost:3000/admin
